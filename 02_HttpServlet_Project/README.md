# HttpServlet Practicals

## Project
02_HttpServlet_Project

## Description
This project contains practical programs using HttpServlet in Advanced Java.  
It demonstrates handling of HTTP GET and POST requests, form processing,  
ServletConfig, ServletContext parameters, and basic calculator operations.

## Package Structure

### com.aoot.http.basic
- Program1: HttpServlet using doGet()
- Program2_post: HttpServlet using doPost()

### com.aoot.http.cal
- Addition: Performs addition of two numbers
- Substraction: Performs subtraction of two numbers

### com.aoot.http.config
- ServletConfigParamServlet: Demonstrates ServletConfig usage
- ServletContextParamServlet: Demonstrates ServletContext usage

## Web Files
- index.html
- post.html
- cal.html
- WEB-INF/web.xml

## How to Run
1. Deploy the project on Apache Tomcat Server.
2. Start the server.
3. Access the respective servlet URL in the browser.
4. Test GET, POST, and Calculator operations.
