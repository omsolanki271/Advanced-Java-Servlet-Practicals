# HttpServlet Practicals

## Project
02_HttpServlet_Project

## Description
This project contains practical programs using **HttpServlet** in Advanced Java.  
It demonstrates:
- Handling HTTP GET and POST requests  
- Form processing  
- ServletConfig and ServletContext parameters  
- Basic calculator operations (Addition, Subtraction, Multiplication, Division)  
- HTTP header handling  
- Registration form processing  

Developed for **MCA Semester 2 – Advanced Java (AOOT Practical).**

---

## Package Structure

### com.aoot.http.basic
- **Program1** – HttpServlet using `doGet()`
- **Program2_post** – HttpServlet using `doPost()`

### com.aoot.http.cal
- **Addition** – Performs addition of two numbers  
- **Substraction** – Performs subtraction of two numbers  
- **Multiplication** – Performs multiplication of two numbers  
- **Division** – Performs division of two numbers  

### com.aoot.http.config
- **ServletConfigParamServlet** – Demonstrates ServletConfig usage  
- **ServletContextParamServlet** – Demonstrates ServletContext usage  

### com.aoot.http.project
- **HeaderServlet** – Demonstrates HTTP header handling  
- **RegistrationServlet** – Demonstrates form data processing

## How to Run
1. Import the project into **Eclipse** (Dynamic Web Project).  
2. Configure **Apache Tomcat Server** (v9 or compatible).  
3. Deploy the project on the server.  
4. Start the server.  
5. Access servlet URLs in the browser.  
6. Test GET, POST, Calculator, and Registration functionalities.
